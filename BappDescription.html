AI Substitutor is a Burp Suite extension that employs AI functionality to alter the values of HTTP request parameters and headers. This extension works efficiently with requests from Swagger / OpenAPI, generated from JavaScript files or those created using the BFScan tool.

Features

This extension uses Artificial Intelligence to automatically substitute the values of HTTP request parameters and headers. It is useful in dealing with requests from Swagger / OpenAPI, JavaScript generated files, and requests generated using the BFScan tool.

Usage

To start using the AI Substitutor extension, make sure you have enabled the use of AI features. Once the AI features are enabled, the extension will automatically substitute the values of HTTP request parameters and headers.
