AI Substitutor is an extension that leverages AI-powered features to automatically substitute HTTP request parameters and headers with contextually relevant values. It is especially useful when working with requests derived from OpenAPI/Swagger definitions, extracted from JavaScript files, or generated via automated tools.

Features

  • AI-assisted substitution of HTTP request parameters and headers
  • Designed to streamline and enhance request manipulation workflows

Usage

  1. Ensure AI-powered features are enabled.
  2. Select part of an HTTP request that contains the substitution content.
  3. Right click, and use the extension-generated context menu item.
  4. Review and modify the substituted values as needed before sending the request.
